# 🚀 PollX

PollX is a modern real-time polling platform where users can create interactive polls, share them instantly, and visualize live results with a beautiful analytics dashboard.

Built with a production-ready full-stack architecture using React, Node.js, Express, PostgreSQL, Drizzle ORM, Socket.IO, and Tailwind CSS.

# ✨ Features

- 🔐 User Authentication (JWT)
- 📊 Create Dynamic Polls
- 🧠 Multiple Questions & Options
- 📈 Real-Time Poll Analytics
- ⚡ Live Result Updates using WebSockets
- 🔗 Shareable Poll Links
- 🌙 Dark / Light Theme
- 📱 Fully Responsive UI
- 🎨 Modern Animated Landing Page
- 🗳 Anonymous Voting Support
- ⏳ Poll Expiration Support
- 🚀 Production Deployment Ready
